# 🚽 Yerevan Codes (Yerevan Toilet Codes) 🇦🇲

> **Живой сайт доступен по адресу:** https://misha12333211-ctrl.github.io/Yerevan-Toilet-Codes/

Народный краудсорсинговый веб-сервис, который помогает жителям и туристам Еревана быстро находить коды от дверей туалетов в кафе, ресторанах и фастфудах. Больше никаких неловких ситуаций и лишних покупок ради чека!

## ✨ Особенности проекта
*   **⚡️ Живой поиск:** Мгновенная фильтрация по названию заведения, адресу и названию ресторана.
*   **🎨 Крафтовый дизайн:** Стильный интерфейс, полностью адаптированный под мобильные экраны и десктопы.
*   **📩 Безбэкендовый сбор данных:** Реализована форма отправки новых кодов пользователями через форму на сайте напрямую на email разработчика без регистрации, баз данных и серверов.
*   **🔋 100% Бесплатно и с открытым исходным кодом:** Проект создан без вложений и хостится бесплатно.

# 🚽 Yerevan Codes (Yerevan Toilet Codes) 🇦🇲

> **Live website:** https://misha12333211-ctrl.github.io/Yerevan-Toilet-Codes/

A crowdsourced web service designed to help residents and tourists in Yerevan quickly find door codes for toilets in cafes, restauramts and fast-food chains across the city. No more awkward situations or unnecessary purchases just for a receipt!

---

## ✨ Features
* **⚡️ Live Search:** Instant filtering by venue name street or restaurant name.
* **🎨 Modern Design:** Modern interface optimized for mobile viewports and desktops.
* **🌐 Trilingual Support (i18n):** Fully localized interface in **Russian, English, and Armenian** to serve the diverse population of Yerevan.
* **📩 Serverless Data Collection:** A form has been implemented allowing users to submit new codes directly to the developer's email via the website, without the need for registration, databases, or servers.
* **🔋 100% Free & Open Source:** Built with zero budget and hosted for free on GitHub Pages.

---

## 🛠 Tech Stack
* **HTML5** — semantic markup.
* **CSS3** — custom fonts, responsive layouts, and UI variables.
* **Vanilla JavaScript** — in-memory data filtering, language switching logic, and DOM manipulation.
